# Leetcode’s Challenges

In this repo I work on Leetcode’s challenges in Javascript.

## Dependencies

`bun` (v1.0 or later)

## Install

`bun install`

## Test

`bun test`

### Debug Tests

`bun test:debug` - Run tests with debugger attached

`bun test:debug:watch` - Run tests in watch mode with debugger attached
